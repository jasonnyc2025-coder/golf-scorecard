# ⛳ 高尔夫记分卡 PWA

18洞高尔夫记分卡，支持安装到手机主屏幕（PWA）。

## 功能
- 18洞记分，−1小鸟 / −2老鹰 / +N博忌
- 实时显示与标准杆差值和总杆数（差值 + 72）
- 每洞可单独清除，支持重置本轮
- 历史记录（带日期）
- 完全离线可用

## 安装到手机

### Android
打开网页后，浏览器会提示"添加到主屏幕"，点击即可安装。

### iPhone / iPad
1. 用 Safari 打开网页
2. 点击底部 **分享** 按钮（方块加箭头图标）
3. 选择 **添加到主屏幕**
4. 点击 **添加**
